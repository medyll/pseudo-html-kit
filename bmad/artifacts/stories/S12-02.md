# S12-02 — Date Picker: Accessible calendar

Points: 3
Priority: Should

## Summary
Implement `date-picker-pk` that enhances native <input type="date"> where available, and provides an accessible popover calendar fallback. Focus on keyboard and screen reader interactions.

## Acceptance Criteria
- Calendar popover is reachable via keyboard and dismissible with Escape.
- Selected date serializes to the component's `data-*` attributes for SSR/CSR handshake.
- Unit tests cover basic selection and keyboard navigation.
