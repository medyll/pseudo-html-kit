## Description

Upgrade component HTML headers for all organisms (13) and frames (20) to use rich JSDoc-style annotations.

- **Organisms**: Currently simple `@props`/`@slots` → upgrade to rich `@prop {type} [name] - description` / `@slot name - description`.
- **Frames**: Currently use `@frame`, `@slots`, `@description` → upgrade to include `@slot name - description`.

**Organisms in scope (13):**
