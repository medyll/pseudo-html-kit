## Description

Implement the `useComponent(url)` React hook inside `pseudo-kit-react`. The hook loads and registers a pseudo-html-kit component by URL, making it available for use as a custom element in JSX.

---

## User Story

As a **React developer**, I want to `import { useComponent } from 'pseudo-kit-react'` and call `useComponent(url)` in a component, so that I can use pseudo-html-kit custom elements directly in JSX without manual registration.

---

## Acceptance Criteria

- [ ] `useComponent(url: string): { ready: boolean }` hook implemented
- [ ] Calls `PseudoKit.register({ name, src: url })` on mount
- [ ] Calls `PseudoKit.init()` after registration
- [ ] Returns `{ ready: false }` until component is registered, then `{ ready: true }`
- [ ] Idempotent: calling with the same URL multiple times is safe
- [ ] `usePseudoKit(urls: string[])` bulk variant also exported
- [ ] Demo app (plain HTML + React 18 via CDN) in `src/pseudo-kit-react/demo/index.html` showing usage
- [ ] Exported from `src/pseudo-kit-react/src/index.js`

---

## Technical Notes

- Name derived from URL filename stem: `button-pk.html` → `button-pk`
- `PseudoKit` imported from `pseudo-stack` (the main package)
- React hook lifecycle: `useEffect` on mount, no cleanup needed (registration is permanent)
- TypeScript types optional for v0.3.0
