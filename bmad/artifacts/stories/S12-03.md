# S12-03 — Color Swatch: Accessible swatch picker

Points: 3
Priority: Could

## Summary
Create `color-swatch-pk` that exposes a small palette of swatches, keyboard-selectable and ARIA-labelled. Useful for theme pickers and palette previews.

## Acceptance Criteria
- Swatches are keyboard navigable and announce selection to screen readers.
- Value is serializable for SSR and reflected in demo pages.
- Unit tests verify selection and accessibility roles.
